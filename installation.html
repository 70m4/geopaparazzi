Installation¶

From Google Play¶

  1. connect your Android device to internet
  2. open the Play Store
  3. type geopaparazzi within the Search textbox
  1. select the first app (not Geopaparazzi 3)
  1. follow installation procedures

Manual Installation¶

  1. from your devices visit the geopaparazzi releases page: https://github.com/geopaparazzi/geopaparazzi Make sure to load the desktop site (not the mobile site).
  2. select the releases tab:
  1. select the latest release apk download link:
  1. the apk installation file will download on the device
  1. and the user will be prompted to open the file:
  1. opening the file will propose the installation:
  1. select install and follow installation procedures
